AuthCore Admin Console

Step A: Register and verify OTP
Step B: Login and capture access token
Step C: Prove protected and admin access
Step D: Manage sessions, users, and roles

Create a user and trigger OTP email delivery.

2. OTP Verification

3. Login & Token

Login to get and store access token for admin actions.

Create and view dynamic roles.

8. Admin User Details
